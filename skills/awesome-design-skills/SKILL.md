# Awesome Design Skills — style registry

67 self-contained visual design styles. Each lives in `styles/<name>/` with:

- `DESIGN.md` — the visual specification (color, type, spacing, motion, signature moves).
- `SKILL.md` — agent-facing instructions for applying that style.

## How to use

1. Match the user's intent (named style, mood, or reference) to a style below.
2. Read `styles/<name>/DESIGN.md` **and** `styles/<name>/SKILL.md` for that style.
3. Apply the spec to the target UI. For structure/UX quality, combine with the
   `ui-ux-pro-max` or `impeccable` skills.

If the user is undecided, show 2–3 candidate styles with a one-line vibe each.
